# MealMate — خوارزمية تصنيف الديناميكي للمطاعم (Basic / Platinum / Elite)

> مرجع تقني مشترك — F03 (تصنيف مطاعم) + F04 (متوسطات التسعير).
> **بدون تدخل بشري** في تصنيف مطعم الفردي — يُعاد الحساب تلقائيًا عند أي تغيير سعر.

> **ملف Excel:** [`MealMate_restaurant_classification.xlsx`](MealMate_restaurant_classification.xlsx)

---

## 1. المبدأ

| المدخل | المعادلة | المخرج |
|--------|----------|--------|
| سعر اشتراك مطعم **26 يوم** (لكل برنامج + باقة) | ÷ 26 | **سعر البوكس اليومي** (د.ك) |
| أسعار يومية لكل مطاعم في **نفس البرنامج والباقة** | μ و σ | **تصنيف** Basic / Platinum / Elite |

> تصنيف **مستقل** لكل تركيبة (برنامج × باقة).

---

## 2. خطوات الحساب

### 2.1 سعر البوكس اليومي

```
سعر البوكس اليومي = سعر الاشتراك الشهري (26 يوم) ÷ 26
```

**مثال (8 مطاعم):**

| مطاعم | شهري (د.ك) | يومي (÷26) | تصنيف* |
|-----|------------|------------|--------|
| مطعم 1 | 83.20 | 3.20 | Basic |
| مطعم 2 | 98.80 | 3.80 | Basic |
| مطعم 3 | 104.00 | 4.00 | Basic |
| مطعم 4 | 117.00 | 4.50 | Platinum |
| مطعم 5 | 122.20 | 4.70 | Platinum |
| مطعم 6 | 143.00 | 5.50 | Platinum |
| مطعم 7 | 156.00 | 6.00 | Elite |
| مطعم 8 | 187.20 | 7.20 | Elite |

\* مع 8 مطاعم تُستخدم الحدود الإحصائية (μ ± 0.5σ).

### 2.2 الإحصائيات

| الرمز | Excel | الشرط |
|-------|-------|-------|
| **μ** | `AVERAGE(C2:C100)` | ≥ 1 مطعم |
| **σ** | `STDEV.P(C2:C100)` | ≥ 2 مطعم |

### 2.3 الحدود

**≥ 2 مطاعم (σ موجود):**

```
الحد السفلي = μ − 0.5 × σ   →  Basic إذا السعر ≤ هذا
الحد العلوي = μ + 0.5 × σ   →  Elite إذا السعر ≥ هذا
                              →  Platinum بينهما
```

**مطعم واحد (σ فارغ) — حدود مرجعية:**

| تصنيف | سعر يومي (د.ك) |
|-------|----------------|
| Basic | **< 4.5** |
| Platinum | **4.5 ≤ سعر < 6** |
| Elite | **≥ 6** |

> أدمن يضبط 4.5 و 6 من الإعدادات — **لا تصنيف يدوي**.

### 2.4 معادلة تصنيف (Excel — العمود D)

```excel
=IF(C2="","",
   IF(AND($F$2<>"", $G$2<>""),
      IF(C2 <= $F$5, "Basic", IF(C2 >= $G$5, "Elite", "Platinum")),
      IF(C2 < 4.5, "Basic", IF(C2 >= 6, "Elite", "Platinum"))
   )
)
```

## 3. أمثلة (سيناريو)

### مطعم واحد — «مطعم الأصيل» 130 د.ك
- يومي = **5.0** → بين 4.5 و 6 → **Platinum**

### مطعمان
- الأصيل: 5.0 | الصحة: 156÷26 = **6.0**
- μ=5.5، σ≈0.707 → سفلي≈5.146، علوي≈5.854
- الأصيل → **Basic** | الصحة → **Elite**

## 4. Excel — `MealMate - تصنيف مطاعم`

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| اسم مطعم | شهري (د.ك) | يومي (د.ك) | تصنيف | | μ | σ |

- **C2:** `=IF(B2="","",B2/26)`
- **F2:** `=IF(COUNT(C2:C100)>0,AVERAGE(C2:C100),"")`
- **G2:** `=IF(COUNT(C2:C100)>1,STDEV.P(C2:C100),"")`
- **F5:** `=IF(AND(F2<>"",G2<>""),F2-0.5*G2,"")`
- **G5:** `=IF(AND(F2<>"",G2<>""),F2+0.5*G2,"")`
- **D2:** معادلة تصنيف أعلاه — اسحب حتى C100

**تنسيق شرطي D:** Basic=أخضر | Platinum=أزرق | Elite=ذهبي

**حدود قابلة للتعديل:** I2=4.5، J2=6 (استبدل 4.5 و 6 في D2 بـ `$I$2` و `$J$2`)

## 5. بعد تصنيف → F04

| اشتراك العميل | متوسط القروب |
|---------------|-----------|
| Basic | متوسط Basic فقط |
| Platinum | متوسط Platinum فقط (بدون Basic) |
| Elite | متوسط Elite فقط |

---

## 6. وصول العميل (هرمي — للتقويم فقط)

Basic → Basic | Platinum → Basic+Platinum | Elite → الجميع

> لا يدخل في μ/σ ولا في متوسط القروب.
