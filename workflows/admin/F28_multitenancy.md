# الأدمن — F28: التوسع الدولي و Multi-Tenancy (Super Admin / Country Admin)

> الدور: الأدمن | الميزة: Multi-Tenancy | القواعد المشتركة: ../00_index_overview.md (القسم 3 و4)

## الهدف
ينظّم هذا التدفق التوسع الدولي عبر طبقتين من الأدمن: **Super Admin** يدير الدول والإعدادات العالمية والرقابة الشاملة، و**Country Admin** يدير دولته فقط، مع عزل تام لبيانات كل دولة.

## المتطلبات المسبقة / المدخلات
- صلاحيات مفصولة: Super Admin (عالمي) مقابل Country Admin (نطاق دولة واحدة).
- إعدادات كل دولة: العملة، بوابات الدفع المحلية، اللغات.
- بنية عزل بيانات تضمن خصوصية وأداء كل دولة.

## خطوات الـ Workflow
1. **Super Admin:** يضيف/يفعّل دولة جديدة ويضبط إعداداتها الأساسية (عملة، لغات، بوابة دفع).
2. **Super Admin:** يعيّن Country Admin لكل دولة ويمنحه صلاحيات نطاقه فقط.
3. **Super Admin:** يراقب تقارير الأداء/الإيرادات/المشتركين لكل الدول أو دولة على حدة، ويضبط الإعدادات العالمية.
4. **Country Admin:** يدير داخل دولته فقط: المطاعم، السائقين، العمولات، الاشتراكات، الشكاوى (كل الميزات F01–F31 ضمن نطاقه).
5. يتأكد الطرفان من عزل البيانات: لا تداخل بين بيانات الدول ولا بين العمليات البنكية.
6. **Super Admin:** يخصّص اللغات/اللهجات لكل دولة (يتقاطع مع F29).

## الحالات والاستثناءات
- محاولة Country Admin الوصول لبيانات دولة أخرى → ممنوعة نظاميًا (عزل صارم).
- إطلاق دولة بلا بوابة دفع/عملة مكتملة → لا تُفعّل للعملاء حتى اكتمال الإعداد.
- تعارض إعداد عالمي مع خصوصية دولة → الإعداد المحلي يُحترم ضمن حدود السياسة العامة.

## القواعد والقيود
- **Super Admin:** نطاق عالمي (كل الدول)، إدارة الدول والإعدادات العالمية والرقابة المجمّعة.
- **Country Admin:** نطاق دولة واحدة فقط، لوحة كاملة مستقلة.
- عزل بيانات الدول تام (خصوصية + أداء).
- تعدد العملات وبوابات الدفع المحلية دون أي تداخل بنكي.

## التكامل مع الأدوار الأخرى
- **المطاعم/السائقون/العملاء:** يُدارون ضمن نطاق دولتهم بواسطة Country Admin.
- **التقارير (F27):** Super Admin يرى المجمّع؛ Country Admin يرى دولته.
- **اللغات (F29) والتغطية (F21):** تُخصّص لكل دولة.

## الإشعارات
- إشعار لـ Country Admin عند تعيينه أو تغيير صلاحياته/إعدادات دولته.
- تنبيهات لـ Super Admin عند مؤشرات حرجة على مستوى أي دولة.
- توثيق أي تغيير في الإعدادات العالمية.
