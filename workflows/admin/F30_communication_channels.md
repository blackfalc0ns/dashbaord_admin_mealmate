# الأدمن — F30: إدارة قنوات التواصل (واتساب/سوشيال)

> الدور: الأدمن | الميزة: Communication Channels | القواعد المشتركة: ../00_index_overview.md (القسم 4.1)

## الهدف
يدير الأدمن قنوات التواصل مع العملاء: زر واتساب العائم للدعم، واستقبال الحالات الاستثنائية (تغيير داخل 72 ساعة)، وروابط السوشيال ميديا، باعتبارها نقطة التماس المباشر للطوارئ والاستفسارات.

## المتطلبات المسبقة / المدخلات
- رقم/حساب واتساب الأدمن للدعم الفني.
- روابط حسابات السوشيال (Instagram، Twitter/X، Snapchat/TikTok).
- سياسة معالجة الحالات الاستثنائية (تتقاطع مع F06).

## خطوات الـ Workflow
1. يضبط الأدمن زر واتساب العائم ليظهر في كل صفحات التطبيق للدعم المباشر.
2. يستقبل عبر واتساب طلبات الحالات الاستثنائية مثل التغيير داخل نافذة 72 ساعة.
3. يتحقق من الطلب وينفّذ الاستثناء من اللوحة عند صحته (راجع F06) ويوثّقه.
4. يجيب الاستفسارات العاجلة ويوجّه العميل للمسار الصحيح.
5. يضبط روابط السوشيال في القائمة الجانبية والتذييل ويحدّثها.
6. يتابع جودة الدعم وزمن الاستجابة كمؤشر خدمة.

## الحالات والاستثناءات
- طلب تغيير داخل 72 ساعة بعد بدء التحضير → يُرفض (راجع F06).
- إساءة استخدام قناة الدعم → توثيق وتعامل وفق السياسة.
- رابط سوشيال معطّل → تحديثه فورًا.

## القواعد والقيود
- واتساب الأدمن هو القناة الرسمية للحالات الاستثنائية والاستفسارات العاجلة.
- استثناءات 72 ساعة تُنفّذ فقط عبر هذه القناة وبتوثيق (راجع F06).
- الاتصال التشغيلي بين العميل والمندوب يبقى داخل التطبيق فقط (راجع F14/F31).
- روابط السوشيال للعرض والتسويق لا للدعم التشغيلي الحرج.

## التكامل مع الأدوار الأخرى
- **العميل:** يستخدم واتساب للدعم والحالات الطارئة، ويرى روابط السوشيال.
- **72 ساعة (F06):** القناة الوحيدة لاستثناءاتها.
- **الخصوصية (F31):** الاتصال داخل التطبيق يظل مقنّعًا ومنفصلًا عن الدعم.

## الإشعارات
- إشعار للأدمن بورود رسائل دعم/طلبات استثناء عبر واتساب.
- إشعار للعميل بنتيجة طلبه الاستثنائي.
