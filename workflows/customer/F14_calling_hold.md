# العميل — F14: الاتصال بالمندوب عند 3 كم

> الدور: العميل | الميزة: الاتصال 3 كم وحالة Hold | القواعد المشتركة: ../00_index_overview.md (القسم 4.8)

## الهدف
تمكين العميل من التواصل مع المندوب بأمان داخل التطبيق عند اقترابه لمسافة 3 كم أو أقل، مع الحفاظ على الخصوصية (أرقام مقنّعة)، وفهم ما يحدث عند عدم الرد (Hold ومحاولة ثانية).

## المتطلبات المسبقة / المدخلات
- طلب نشط قيد التوصيل وتتبع لحظي مفعّل (راجع F13).
- وصول المندوب إلى نطاق 3 كم أو أقل.
- صلاحية الميكروفون/الاتصال داخل التطبيق عند اللزوم.

## خطوات الـ Workflow
1. قبل 3 كم: زر الاتصال مخفي/Disabled مع رسالة توضيحية.
2. عند 3 كم أو أقل: يظهر **زر الاتصال** في شاشة التتبع مع تنبيه الخصوصية.
3. يضغط العميل للاتصال بالمندوب **داخل التطبيق** (رقم مقنّع، رسائل مشفّرة).
4. إن لم يرد العميل على المندوب: يسجّل المندوب محاولة فاشلة ويتحول الطلب إلى **Hold**.
5. يعود المندوب للمنطقة ويجري **محاولة ثانية**، ويصل العميل إشعار بذلك.
6. عند فشل الثانية: يُظهر النظام رقم العميل للمندوب كاستثناء موثّق مع إشعار الأدمن.

## الشاشات والعناصر
- **الاتصال بالمندوب:** زر اتصال، حالة الاتصال، تنبيه الخصوصية.
- **التتبع اللحظي:** ظهور/إخفاء زر الاتصال حسب المسافة.
- **Timeline تفاصيل الطلب:** تسجيل كل أحداث الاتصال وHold.

## الحالات والاستثناءات
- **Disabled (قبل 3 كم):** رسالة «الاتصال يتاح عند الاقتراب من العميل».
- **Hold:** عند عدم الرد يُعلّق الطلب مؤقتًا ويكمل المندوب توصيلاته.
- **محاولة ثانية:** إشعار جديد للعميل عند عودة المندوب.
- **إظهار الرقم:** حالة استثنائية بعد فشل محاولتين + إشعار الأدمن + تسجيل الحدث.
- **Offline/Permission:** توضيح سبب الصلاحية قبل نافذة النظام.

## القواعد والقيود
- زر الاتصال يظهر فقط عند 3 كم أو أقل (في تطبيقي العميل والمندوب).
- الاتصال داخل التطبيق فقط، بأرقام مقنّعة ورسائل مشفّرة.
- إظهار الرقم الحقيقي استثناء موثّق فقط بعد فشل محاولتين وتحويل Hold.
- كل الأحداث تُحفظ في Timeline تفاصيل الطلب.

## التكامل مع الأدوار الأخرى
- **المندوب:** يبدأ الاتصال، يسجّل المحاولات، يحوّل إلى Hold، ويجري المحاولة الثانية.
- **الأدمن:** يصله إشعار عند إظهار الرقم ويتابع الحدث في تفاصيل الطلب.
- **المطعم:** لا تكامل مباشر في هذه المرحلة.

## الإشعارات
- إشعار عند إتاحة الاتصال (وصول 3 كم).
- إشعار بمحاولة التواصل الثانية بعد Hold.
- إشعار للأدمن (وتسجيل) عند إظهار رقم العميل استثناءً.
